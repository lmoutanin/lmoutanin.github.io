Contexte du projet



Le but du projet était de concevoir et de mettre en place un système de gestion informatisé afin de moderniser et centraliser les opérations quotidiennes du garage 97CAR. Ce système visait à optimiser la gestion des véhicules, des propriétaires, des réparations ainsi que des factures, dans le but d'améliorer l'efficacité, la traçabilité et la qualité du service au sein de l’entreprise.


Missions Principale :

  • Analyse des besoins : étude des processus existants à partir des documents fournis (factures, fiches véhicules, listes clients).
  • Conception de la base de données : modélisation des entités (voiture, propriétaire, réparation, facture) et des relations.
  • Installation de l’infrastructure réseau : configuration d’un serveur web Apache, d’un serveur MySQL, d’un DNS, et d’un routeur sécurisé.
  • Développement de l’application web :
    1. Création de la base de données MySQL
    2. Développement des scripts PHP pour la gestion des données
    3. Mise en place d’une interface utilisateur (HTML/PHP)
    4. Création de formulaires dynamiques
    5. Génération automatique de factures
    6. Implémentation de la gestion de sessions pour sécuriser l’accès
  • Tests, corrections et documentation technique
  • Conception d’une charte graphique et d’un logo

Environnement technique :

  • Langages utilisés : PHP, HTML, SQL, CSS
  • Base de données : MySQL (via PhpMyAdmin)
  • Serveur Web : Apache 2 (Linux Debian)
  • Contrôle de version : Git
  • Virtualisation : Hyper-V
