Description du projet


Dans ce projet, une interface d'authentification a été mise en place avec Symfony pour permettre aux utilisateurs de se connecter à l'interface d'administration du garage et d'accéder aux outils disponibles.​




Dans le cadre du projet 97CAR, quatre outils principaux ont été développés pour faciliter la gestion quotidienne du garage :


1. Client

Cette interface permet l'enregistrement et le suivi des informations des clients.​






2. Voiture

Cette interface permet de gère les détails des véhicules, y compris leur association avec les clients.​






3. Réparation

Cette interface permet d'enregistre les interventions réalisées sur les véhicules, avec les détails correspondants.​​





4. Facture

Cette interface permet de génère et archive les factures liées aux interventions effectuées.​








Autre

Dans ce projet, un système de déconnexion a été mis en place avec Symfony pour permettre aux utilisateurs de se déconnecter de l'interface d'administration du garage. Cette fonctionnalité assure la sécurité en mettant fin à la session active, empêchant ainsi tout accès non autorisé aux outils de gestion.
