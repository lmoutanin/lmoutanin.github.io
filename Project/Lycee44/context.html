Contexte du projet



Le Lycée 44 est une école innovante spécialisée dans les technologies du numérique et de l’intelligence artificielle. Elle propose un enseignement basé sur des projets concrets en mode collaboratif, sans cours magistraux. Dans cet environnement autonome, la gestion des absences est un enjeu important. Pour répondre à ce besoin, le lycée a souhaité développer une application web de suivi des absences.


Missions Principale :

  • Développement de l’application avec le framework Symfony.
  • Mise en place d’un système d’authentification pour les enseignants.
  • Création d’une interface permettant de signaler une absence avec date et matière.
  • Fonctionnalités avancées : Authentification sécurisée, gestion des rôles, et dashboard interactif.
  • Déploiement : Hébergement de l’application sous Docker sur une machine virtuelle Linux.
  • Développement d’un tableau de bord pour la consultation des absences par les responsables.
  • Documentation : Rédaction des guides techniques et d’utilisation.
  • Développement d’un tableau de bord pour la consultation des absences par les responsables.
  • Conception graphique : Création du logo

Environnement technique :

  • Langages et Frameworks : PHP (Symfony)
  • Base de données : PostgreSQL
  • Outils de virtualisation : Docker
  • Contrôle de version : Git

Compétences Acquises:


  • Maîtrise du framework Symfony et du modèle MVC
  • Développement d’une application web sécurisée avec gestion d’authentification
  • Programmation : Utilisation de PHP avec Symfony pour développer des applications robustes.
  • Gestion de bases de données : Conception et administration de bases PostgreSQL pour une gestion efficace des données.
  • Conception d’une base de données relationnelle
  • Utilisation de Git pour le versioning de projet
  • Déploiement : Installation et configuration de l'application sur une VM Linux avec Docker pour une infrastructure fiable.
