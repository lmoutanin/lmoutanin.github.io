Contexte du projet

Le but du projet était de concevoir et de mettre en place un système de gestion informatisé afin de moderniser et centraliser les opérations quotidiennes du garage 97CAR. Ce système visait à optimiser la gestion des véhicules, des propriétaires, des réparations ainsi que des factures, dans le but d'améliorer l'efficacité, la traçabilité et la qualité du service au sein de l’entreprise.


Missions Principale :

  • Analyse des besoins : Collaboration avec l’équipe pour définir les exigences fonctionnelles et techniques.
  • Modélisation de la base de données : Conception sous PostgreSQL pour gérer les véhicules, propriétaires et interventions.
  • Développement web : Création d’une interface avec Symfony pour gérer les véhicules, réparations et factures.
  • Fonctionnalités avancées : Authentification sécurisée, gestion des rôles, et dashboard interactif.
  • Déploiement : Hébergement de l’application sous Docker sur une machine virtuelle Linux.
  • Gestion de projet : Organisation en sprints selon la méthode Agile (Scrum).
  • Documentation : Rédaction des guides techniques et d’utilisation.
  • Conception graphique : Création du logo et de la charte graphique du garage.

Environnement technique :

  • Langages et Frameworks : PHP (Symfony)
  • Base de données : PostgreSQL
  • Outils de virtualisation : Docker
  • Contrôle de version : Git
  • Méthodologie de gestion de projet : Scrum

Compétences Acquises:


Développement d'applications web :

  • Analyse et conception fonctionnelle : Recueil et définition des besoins utilisateurs pour une solution adaptée
  • Programmation : Utilisation de PHP avec Symfony pour développer des applications robustes.
  • Gestion de bases de données : Conception et administration de bases PostgreSQL pour une gestion efficace des données.

Gestion de projet agile :

  • Planification et suivi : Application de la méthodologie Scrum pour une gestion efficace du projet.
  • Collaboration : Travail d'équipe pour assurer la réussite du projet.

Administration système :

  • Déploiement : Installation et configuration de l'application sur une VM Linux avec Docker pour une infrastructure fiable.
