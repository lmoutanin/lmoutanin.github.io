Base de données


Lors de mon premier Project, j'ai conçu une modélisation de base de données pour le garage 97CAR, qui servira de point de départ pour ce projet. Cette modélisation comprend les entités principales suivantes :


Clients :

  • Stocke les informations personnelles des clients, telles que le nom, l'adresse, le numéro de téléphone et l'email.

Véhicules :

  • Contient les détails des véhicules entretenus, y compris la marque, le modèle, l'année, le numéro d'immatriculation et l'ID du propriétaire, établissant ainsi une relation avec l'entité Clients.

Réparations :

  • Enregistre chaque intervention effectuée sur les véhicules, avec des informations telles que la date de la réparation, la description des travaux effectués, le coût associé et l'ID du véhicule concerné, créant une liaison avec l'entité Véhicules.

Factures :

  • Gère les données relatives à la facturation des réparations, incluant le numéro de facture, la date d'émission, le montant total et l'ID du client, assurant une connexion avec l'entité Clients.
